### Hình demo bài làm:
Khi user mở app, splash screen xuất hiện --> user click bất kì trên màn hình --> truy cập vào màn hình login.
- Màn hình login:<br/>
<img width="334" height="715" alt="image" src="https://github.com/user-attachments/assets/12c55dd2-b7d8-4ae2-b918-ce3191ffbcfc" /><br/>
  Khi user click dòng TextView "<b>Are you new user? Register</b>" --> truy cập vào màn hình register.<br/>
- Màn hình register (sign up):<br/>
  <img width="341" height="728" alt="image" src="https://github.com/user-attachments/assets/dbab9f61-56ae-4d18-a45a-7a7fd355302d" /><br/>

Nếu user ở màn hình register, muốn quay lại màn hình login thì bấm nút Back của điện thoại.<br/>
Trong suốt quá trình sử dụng app, splash screen chỉ xuất hiện duy nhất một lần khi user khởi động app lần đầu.
